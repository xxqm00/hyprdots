## Requirements

+ Arch Linux system (Made on hyprland, 0.55)
+ Listed applications below
+ _HIGH_ emphasis on yay, as I use it for most of my applications, but do read the PKGBUILDS before installing anything in light of recent malware attacks on the AUR.

### Applications
+ [yay](https://github.com/Jguer/yay)
+ flatpak
+ kitty (terminal)
+ firefox (browser)
+ waypaper (wallpaper switcher)
  +  awww for wallpaper management (gifs, pngs, etc)
+ vicinae (app launcher, among _other_ things)
+ thunar (file manager)
+ veila (screen locking manager)
+ spotify (flatpak version is configured, can be changed in binds)

### Installation
- clone or download this repository as a zip file, and drag the folders that you want into your .config folder
OR
- Use the bash installer script to download the listed applications
